## 项目简介

LearnMate 是一个面向长期备考人群的 AI 学习工作台。它不是另一个内容生成工具，而是把用户每天的学习过程组织成**可执行的闭环**——从情绪启动、苏格拉底式追问、资料解析到间隔复习，让 AI 真正成为学习流程的搭子。

**核心主张**：不堆内容，只组织流程。

## 核心功能

### 1. 今日学习闭环
系统每天给出 3 个以内的任务，确保用户能在短时间内完成一个完整的学习循环。

### 2. 多 Agent 协作
- **Star**：情绪陪伴，降低启动成本
- **Ask**：苏格拉底式追问，推动深度理解
- **Mem**：间隔复习，根据遗忘曲线安排
- **Data**：弱点分析，定位错题模式

### 3. 资料到知识节点
输入学习材料，自动提取知识节点，生成练习题、复习计划和知识图谱。

### 4. 个人知识图谱
将知识节点连成可视化图谱，展示概念之间的关系。

## 演示路径（90 秒上手）

1. **今日页** → 提交挑战回答，体验 Ask 追问
2. **搭子页** → 切换 Agent，体验多角色协作
3. **资料页** → 粘贴材料，生成知识节点
4. **评测页** → 查看评测指标与隐私声明

---

## 评测标准

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 首次上手时间 | ≤90 秒 | 完成一次挑战或解析 |
| 每日任务完成率 | ≥1 个/天 | 平均完成 1 个以上任务 |
| 知识节点准确性 | ≥4/5 分 | 人工抽检节点质量 |
| 追问质量 | 推动理解 | 一次只问一个问题 |
| 闭环完整度 | 4 项齐全 | 节点+练习+复习+图谱 |
| 降级可用性 | 无需 API | 本地规则引擎完整演示 |

## 目标用户

- 雅思、考研、公考等长期备考者
- 有学习资料和目标，但缺少执行系统的人
- 容易因焦虑、摆烂、卡住而中断学习的人
